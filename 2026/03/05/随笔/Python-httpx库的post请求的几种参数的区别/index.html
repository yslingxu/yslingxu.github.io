Python-httpx库的post请求的几种参数的区别

发表于2026-03-05|更新于2026-03-05|随笔
|浏览量:

1. data

将参数转换成了&连接的格式。

2. json

请求体为JSON字符串，且在HTTP头中标注为application/json格式。

3. content

请求体为JSON字符串，但是并未在HTTP头中标注格式。
